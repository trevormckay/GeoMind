 SYNTAX 
        To find an equation: eq <find> <figure> 
        To find a value: <given> <given 2 if it applies> <figure> 
      
        SUPPORTED FIGURES 
        Spheres 
        Cylinders (Partially) 
      
        ABBREVIATIONS 
        vol = volume 
        sa = surface area 
        rad = radius 
        b = area of a base 
        la = lateral area 
        ta = total area
